# Gear Pair Calculator — 仕様書

## 対象

- 平歯車2枚組（外歯車）
- インボリュート歯形
- 同一モジュール・同一圧力角のかみ合い

---

## 1. 扱う数値

### 入力可能な値

| 記号 | 名称 | 単位 | 備考 |
|------|------|------|------|
| `m`  | モジュール | — | 正の実数 |
| `z1` | 歯車1の歯数 | — | 正の整数 |
| `z2` | 歯車2の歯数 | — | 正の整数 |
| `α`  | 圧力角 | deg | デフォルト 20°、入力省略可 |
| `a`  | 中心距離 | mm | Mode 3 で入力 |
| `i`  | ギア比 | — | Mode 2 で入力 |

### 出力される値（計算結果）

| 記号 | 名称 | 対象 |
|------|------|------|
| `d`  | ピッチ円直径 | 各歯車 |
| `da` | 外径（歯先円直径） | 各歯車 |
| `df` | 歯底円直径 | 各歯車 |
| `db` | 基礎円直径 | 各歯車 |
| `p`  | 円ピッチ | 共通 |
| `a`  | 中心距離 | 歯車組 |
| `i`  | ギア比 | 歯車組 |

---

## 2. 計算式

標準平歯車（転位なし）の計算式を使用する。

| 項目 | 計算式 |
|------|--------|
| ピッチ円直径 | `d = m × z` |
| 外径 | `da = m × (z + 2)` |
| 歯底円直径 | `df = m × (z − 2.5)` |
| 基礎円直径 | `db = m × z × cos(α)` |
| 円ピッチ | `p = π × m` |
| 中心距離 | `a = m × (z1 + z2) / 2` |
| ギア比 | `i = z2 / z1` |

> 歯末のたけ（addendum）= 1m、歯元のたけ（dedendum）= 1.25m（標準値）

---

## 3. 入力モード

### Mode 1: 歯数指定

入力: `m`, `z1`, `z2`（`α` は省略時 20°）

出力: `d1`, `d2`, `da1`, `da2`, `df1`, `df2`, `db1`, `db2`, `p`, `a`, `i`

### Mode 2: ギア比指定

入力: `m`, `z1`, `i`（`α` は省略時 20°）

計算: `z2 = round(z1 × i)`（整数に丸める）

出力: `z2`（丸め後）、各寸法、実際の `i`、`a`

### Mode 3: 中心距離指定

入力: `a`, `z1`, `z2`（`α` は省略時 20°）

計算: `m = 2a / (z1 + z2)`

出力: `m`、各寸法、`i`

---

## 4. エラー条件

| 種別 | 条件 |
|------|------|
| エラー | 入力欄が空欄 |
| エラー | 入力値が 0 以下 |
| エラー | 歯数が整数でない |
| エラー | 計算不能な組み合わせ（例: `z1 + z2 = 0`） |

---

## 5. 警告条件

| 種別 | 条件 |
|------|------|
| 警告 | 歯数が少ない（`z < 17`、圧力角 20° の場合のアンダーカット基準） |
| 警告 | Mode 2: `z2` の丸め前の値が整数から 0.1 以上ずれている |
| 警告 | Mode 3: 求めた `m` が JIS 標準モジュール値（1, 1.25, 1.5, 2, 2.5, 3, 4, 5, 6, 8, 10, 12, 16, 20 …）から外れる |

---

## 6. 技術スタック

| 項目 | 選択 |
|------|------|
| 動作環境 | ブラウザ（HTML/CSS/JS） |
| グラフィック | Canvas または SVG（v0.3 以降） |
| ビルドツール | なし（バニラJS で開始） |
| ファイル構成 | `index.html` / `calc.js`（計算ロジック）/ `ui.js`（画面処理） |

計算ロジック（`calc.js`）と画面処理（`ui.js`）は分離する。
