# Gear Pair Calculator

インボリュート歯形の平歯車2枚組について、一部の数値を入力すると残りの寸法やギア比を計算するブラウザアプリ。

## 起動方法

`<script type="module">` を使用しているため、**ファイルを直接ブラウザで開くだけでは動作しません**。ローカルサーバー経由で開いてください。

**VS Code の Live Server を使う場合:**

1. 拡張機能 "Live Server" をインストール
2. `index.html` を右クリック → "Open with Live Server"

**Python でサーバーを起動する場合:**

```bash
cd path/to/Gear
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開く。

## 機能（v0.2 現在）

- Mode 1: `m`, `z1`, `z2` を入力して計算ボタンを押すと全寸法を表示
- Mode 2: `m`, `z1`, `i` を入力すると `z2` を算出し、ギア比誤差も表示
- エラー入力（空欄・0以下・小数の歯数）はエラーメッセージで通知

## 入力モード

| モード | 入力 | 求まるもの | 状態 |
|--------|------|-----------|------|
| Mode 1: 歯数指定 | `m`, `z1`, `z2` | 全寸法、`i`, `a` | ✅ v0.1 |
| Mode 2: ギア比指定 | `m`, `z1`, `i` | `z2`（丸め）、全寸法、ギア比誤差、`a` | ✅ v0.2 |
| Mode 3: 中心距離指定 | `a`, `z1`, `z2` | `m`、全寸法、`i` | 未実装 |

## 確認済み入力例

### Mode 1: 歯数指定

| 入力 | 値 |
|------|----|
| モジュール m | 2 |
| 歯数 z1 | 20 |
| 歯数 z2 | 40 |

| 結果項目 | 値 |
|----------|----|
| ギア比 i | 2.0000 |
| 中心距離 a | 60.000 mm |
| 円ピッチ p | 6.283 mm |
| ピッチ円直径 d（歯車1） | 40.000 mm |
| ピッチ円直径 d（歯車2） | 80.000 mm |

### Mode 2: ギア比指定（端数なし）

| 入力 | 値 |
|------|----|
| モジュール m | 2 |
| 歯数 z1 | 20 |
| 目標ギア比 i | 2 |

| 結果項目 | 値 |
|----------|----|
| 理想 z2 | 40.0000 |
| 丸め後 z2 | 40 |
| 実際のギア比 | 2.0000 |
| ギア比誤差 | +0.0000 |
| 中心距離 a | 60.000 mm |
| 円ピッチ p | 6.283 mm |

### Mode 2: ギア比指定（端数あり）

| 入力 | 値 |
|------|----|
| モジュール m | 2 |
| 歯数 z1 | 21 |
| 目標ギア比 i | 2.3 |

| 結果項目 | 値 |
|----------|----|
| 理想 z2 | 48.3000 |
| 丸め後 z2 | 48 |
| 実際のギア比 | 2.2857 |
| ギア比誤差 | -0.0143 |
| 中心距離 a | 69.000 mm |
